

# Kodi Chandu – Portfolio Website

## Overview
A modern, visually striking portfolio website for a Full Stack Web Developer featuring smooth animations, glassmorphism design elements, multi-color gradients (dark base with blue, purple, cyan, orange accents), and a dark mode toggle.

---

## Pages & Sections (Single-Page Layout with Smooth Scroll)

### 1. Sticky Navbar
- Transparent/glassmorphism navbar that becomes solid on scroll
- Navigation links: About, Skills, Experience, Projects, Certifications, Contact
- Dark mode toggle button
- Mobile hamburger menu

### 2. Hero Section
- Large animated heading: **Kodi Chandu**
- Subtitle: *Software Developer | Full Stack Web Developer*
- Tagline with typing or fade-in animation
- Two CTA buttons: "View Projects" (scrolls down) and "Download Resume" (placeholder link)
- Social icons: LinkedIn, LeetCode, Email
- Animated developer illustration/SVG background

### 3. About Me
- Professional summary paragraph
- Animated developer-themed SVG/illustration
- Fade-in scroll animation

### 4. Skills Section
- Categorized skill cards with animated progress bars
- **Languages & Frameworks:** Java, JavaScript, React JS, React Native, Next.js, Node.js, Express.js
- **Databases:** PostgreSQL
- **Concepts:** Data Structures & Algorithms, REST APIs
- Cards animate in on scroll

### 5. Tools & Technologies
- Animated icon grid: VS Code, GitHub, Postman, Android Studio, pgAdmin
- Hover scale/glow effects on each tool card

### 6. Experience (Timeline)
- Vertical timeline layout
- **Software Developer – Pivotal Soft IT Solutions** (July 2023 – Present)
- Bullet points for responsibilities
- Animated timeline entry on scroll

### 7. Projects (Card Layout)
- **Care Connect** – Doctor Appointment App (React, Node.js, PostgreSQL)
- **House Price Prediction** – ML-based prediction tool
- Each card: hover lift effect, tech stack tags, "View Project" button
- Fade/scale-in animations

### 8. Certifications
- Clean card or badge layout
- Java Full Stack – TAP Academy
- ML Master Class – Pantech/APSSDC

### 9. Contact Section
- Contact info: email, phone, LinkedIn, LeetCode
- Animated contact form (name, email, message) with success toast notification
- Note: form will be frontend-only (no backend email sending without a backend integration)

### 10. Footer
- Social links, copyright, quick nav links

---

## Design & UX
- **Color palette:** Dark background with vibrant blue, purple, cyan, and orange accents
- **Glassmorphism** cards and navbar with backdrop blur
- **Gradient sections** for visual variety
- **Smooth scroll** between sections
- **Fade-in, scale-in, slide-in** animations on scroll using Intersection Observer
- **Animated skill bars** filling on visibility
- **Dark/Light mode** toggle with smooth transition
- **Fully responsive** for desktop, tablet, and mobile

## Technical Notes
- No backend needed – this is a static portfolio site
- Contact form will show a success animation but won't actually send emails (can be added later with a backend)
- Resume download will use a placeholder link
